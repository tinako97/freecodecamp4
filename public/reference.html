API Basejump: Timestamp microservice

by tinako97

Pass either a Unix timestamp or a natural language date. Receive a JSON object of that date in both unix and natural language.

Example usage:

https://freecodecamp4-tinako97.c9users.io/December 15, 2015

https://freecodecamp4-tinako97.c9users.io/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
